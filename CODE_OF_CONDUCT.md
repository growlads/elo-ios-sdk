# Code of Conduct

This project adopts the [Contributor Covenant, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

By participating in this project — opening issues, submitting pull requests, or otherwise interacting in project spaces — you agree to abide by its terms.

## Reporting

To report a concern, email **conduct@elo.ad**. Reports are handled privately.
