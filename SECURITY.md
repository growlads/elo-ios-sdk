# Security policy

If you believe you have found a security vulnerability in the Elo iOS SDK or in this repository, please report it privately rather than opening a public issue.

**Email:** security@elo.ad

We will acknowledge receipt within 3 business days and aim to provide a substantive response within 10 business days.

When reporting, please include:

- A description of the vulnerability and its potential impact.
- Steps to reproduce, ideally with a minimal code sample.
- The SDK version and iOS version you tested against.

We disclose fixes via the [CHANGELOG](./CHANGELOG.md) once a patched release is tagged on this repository.
