# Contributing

Thanks for your interest in the Elo iOS SDK. This repository is the **distribution** for the SDK — the XCFramework binary, the generated `Package.swift`, and the AdMob mediation source. The SDK itself is developed in a separate (private) source repository, so the kinds of changes accepted here are narrow.

## Reporting bugs

Please open an issue using the **Bug report** template. Include the SDK version, iOS version, device, and a minimal repro.

For integration help, use the **Integration question** template instead.

## Security issues

Do **not** open a public issue. See [SECURITY.md](./SECURITY.md).

## Pull requests

PRs are welcome for the editable surface of this repo:

- `README.md` — consumer-facing integration docs.
- `Example/` — the runnable integration sample.
- `LICENSE`, `.gitignore`, `CHANGELOG.md`, this file, and `.github/`.

Please **do not** edit the following — they are regenerated by the publish workflow and your changes will be clobbered on the next release:

- `Package.swift` (the `binaryTarget` URL and checksum are computed during the XCFramework build).
- `Sources/GrowlAdsMediationAdMob/` (rsynced from the source repo).

If you want to fix something inside the SDK itself, file a bug here and we'll patch it upstream and re-publish.
