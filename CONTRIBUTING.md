# Contributing

Thanks for your interest in the Elo iOS SDK. This repository is the **distribution** for the SDK — the XCFramework binary and the generated `Package.swift`. Mediation adapters (e.g. AdMob) ship from the separate [`growlads/elo-ios-mediation`](https://github.com/growlads/elo-ios-mediation) package. The SDK itself is developed in a separate (private) source repository, so the kinds of changes accepted here are narrow.

## Reporting bugs

Please open an issue using the **Bug report** template. Include the SDK version, iOS version, device, and a minimal repro.

For integration help, use the **Integration question** template instead.

## Security issues

Do **not** open a public issue. See [SECURITY.md](./SECURITY.md).

## Pull requests

PRs are welcome for the editable surface of this repo:

- `README.md` — consumer-facing integration docs.
- `Example/` — the runnable integration sample.
- `LICENSE`, `.gitignore`, `CHANGELOG.md`, this file, and `.github/`.

Please **do not** edit `Package.swift` — it is regenerated by the publish workflow and your changes will be clobbered on the next release. The `binaryTarget` URL and checksum are computed during the XCFramework build in the source repo.

If you want to fix something inside the SDK itself, file a bug here and we'll patch it upstream and re-publish.
